Se você respondeu "sim" para uma das perguntas acima este é o curso ideal para você.

Este curso é indicado para pessoas que querem ter o primeiro contato com programação de computadores, porém não tem paciência para longas e maçantes aulas teóricas, aqui teremos prática de forma simples e direta com o que realmente é relevante para iniciantes ou estudantes na área de programação e até alguns exemplos simples e práticos de aplicação.

OBS: Curso introdutório, indicado como primeiro contato com a Lógica de Programação não é recomendado para pessoas que já programam ou que já conhecem raciocínios lógicos aplicados em programação.
